CCBN 主题演讲 | 自研力铸造数智网络，服务数字中国建设
发布日期：
2023-04-21
浏览次数：

第29届中国国际广播电视信息网络展览会（CCBN）已在北京首钢会展中心正式拉开帷幕，现场汇聚了政府领导、行业专家、业界精英以及众多产业生态链合作伙伴，震有科技副总裁桂新华带着“专注自研，服务客户”的发展理念，以《自研力铸造数智网络——服务数字中国建设》为题在主题报告会上发表演讲，分别从“数智网络深融合”、“震有科技两大自研力”、“创新应用落地”向大家全方位展示了震有科技如何以自研力筑基数智网络，助力广电业务创新。


演讲全文

尊敬的各位嘉宾、专家，大家好！很荣幸在 CCBN 大会上与大家分享我们公司的想法和实践，首先请允许我简单介绍一下震有科技。震有是一家提供综合通信解决方案的企业，秉持“专注自研，服务客户”的发展理念，希望通过今天的分享，让大家了解我们的自研力。





90 年代初，电话交换机是进口国外的设备，一线价格是 3000 元左右；90 年代末，国产化之后，一线价格是 300 元...所以自研力是最有效率的生产力，也是一家科技企业的根本。今天我演讲的主题是“自研力铸造数智网络，服务数字中国建设”，与大家一起探讨三个问题，在数字经济的大趋势下，网络将如何发展？自研力如何驱动数智网络建设？以及广电的数智网络可以有哪些创新？




# 瞄准“深融合”，数智网络发展新趋势 #


数字化是热点，也是我们一直坚持在走的路。网络的数智发展，我认为要瞄准“深融合”方向。一方面，云技术和AI技术催生了新一轮的应用革命，云边端协同的新网络与多种媒体、应用能力的深融合势不可挡。另一方面，加快建设高质量的5G网络和千兆光网，打通数字基础设施大动脉，促进行业网络的深融合势在必行。


在5G网络方面，5G经过这几年的发展，在全球部署已超过200张，而中国拥有全球最大的5G网络，其中专网覆盖超过3000张。5G工业专网极大的促进了工业数字化转型，5G 发展也逐步进入深水区。今年年中 Rel17 冻结，其中多个关键技术，如 TSN、RedCap、NTN、定位、D2D 增强等，涉及到 5G 工业级组网、5G物联网、车联网、低轨卫星等多个热门领域，需要我们持续推动产业落地开花结果。同时 5.5G 的立项，真正为我们带来了 5G 确立时的初心——10Gbps、天地一体全连接等等。震有也在积极推动5G的技术研发，投入开发 Rel17 特性并测试部署落地，预研 5.5G 标准，推动产业生态发展，为合作伙伴提供更多创新选项。



在媒体深融合方面，主要的发力点集中在网络融合与科技融合上。首先在传统的三网融合基础上，加快5G无线接入、F5G光网固定接入、卫星天地接入等多种接入方式的建设，融合触及更广的终端用户，实现全覆盖；其次，利用媒体云结合大规模AI应用、AR/VR、4K/8K等能力，满足更丰富的业务需求。核心网IMS是传统语音的业务通道，在“深融合”上具备独特的商业与社会价值，随着3GPP数据通道的增加，它已成为5G新通道实现融媒体创新的绝佳渠道。IMS 已具备调用融媒体云能力的特性，实现商务、协作、媒体场景的创新，既能增强运营商码号的核心价值，也能促进触及到更多终端用户。


在F5G方面，聚力双千兆城市建设是重点。首先运营商要求的高效、精细、智能化运营变革，随着SDN的发展，开放解耦技术促进了统一NETCONF/YANG 接口的定义，多厂商设备统一运营的难题得以解决；大数据、AI技术的应用，也促进了光网络的智慧运营。其次对于2H业务，从GPON 到 10G PON并向 50G PON 发展演进，速率和体验上都有极大的提升；2B业务，政企客户的网络组成由传统以太网交换机向 F5G 全光网络演进，实现光网络基座的“一次投入，无限演进”，网络层次由“城域+接入” 向“城域接入”演进，通过 OTN-CPE 实现一跳入云。向50G PON 的演进中，由 5G 宏站覆盖/重点接入到 5G 高频毫米波基站/全面覆盖的转变，使得 FTTM 成为可能。


# 自研力是铸造数智网络的基石 #

网络的数智发展需要依靠5G端到端网络、F5G光网络的创新应用和发展，推动数智网络建设的落地需要依靠自研力。震有可以提供5G端到端核心网、F5G光网两大基础自研力。


首先是端到端核心网，震有于 2005 年，从Soft Switch 开始研发，获得了欧洲部分项目；2009 年跟随 3GPP 标准向 IMS 演进，开发的信令产品在印度部署；2013 年震有开始投入研发 NFV 的虚拟化解决方案，IMS 产品在国内和海外实现多点部署。2019 年，震有开拓分组域核心网产品，开发了基于 Cloud Native 的云化解决方案，在卫星领域赢得了中国电信的高轨卫星天通一号项目。2020 年震有向 5G+演进，开发了 5G RCS，实现了公有云业务部署，基于智能网卡提升解决方案等。震有形成了从专用硬件（ATCA）到云化部署的全套自研产品，含 IMS、EPC/5GC、信令处理、5G 消息等端到端解决方案，核心网专利已超 50 项、软著 75 项、全球用户超 6000 万。


UnionCore 是震有开发的深度融合核心网产品，提供 4G/5G 全融合、全云化、端到端的分组核心网。NuMax Cloud 为每个 4G 和 5G 核心网网元提供集中式统一的端到端网管。我们的融合核心网产品基于 ETSI NFV 标准打造，可部署在各种私有云之上，如 RedHat Enterprise Linux；也支持在公有云部署，如阿里云。在服务器一侧，浪潮、IBM、新华三等都是我们的合作伙伴。 


震有提供云化统一IMS 产品，从接入层到控制层设备，再到上层的语音应用，全系列 IMS 产品。在国内震有是少数能提供全系列 IMS 解决方案的厂家，支持任意语音接入方案，如早期基于 E1/T1 的 PRI、V5、R2 窄带语音，支持基于 IP 的 MGCP、SIP 宽带语音，支持 4G 的 VoLTE，5G 的 VoNR，也支持最新的 VoWiFi，可提供多媒体会议/电话、即时消息、彩铃彩信、IP 传真、5G 消息等通信服务。


震有科技的光网络自研力值得信赖。首先团队核心成员均来自一流的通信公司，产品开发经历丰富；从2005-2022年完成了光电一体化、WDM/GPON、DCI/M-OTN/10G PON等一系列光网络产品的自研和更新，形成了完整的光网络解决方案，是国内率先推出全自研 GPON、XG-PON、XGS-PON 以及 Combo PON 的原创厂家之一。今年正在研发 50G PON OLT 产品、基于 OB-BAA 方向的 VOLT 产品、SDN Controller 产品，以及带 DPI 功能的产品。


2022 年震有完成了对杭州晨晓公司、依赛通信公司的收购，可提供 SDH/MSTP、PTN、WDM 多种传输技术，以及覆盖核心层、汇聚层和接入层的全光网络传输产品系列，为用户提供高可靠、有 QoS 保证、可运营、可管理的光传送网。


在光接入网，GX3500-S 系列框式 OLT 和 GX3100 系列盒式 OLT 融合下一代 PON 技术，提供高密度的 GPON、XG-PON、XGS-PON和 Combo PON，上联支持 100G 等各种速率，配合丰富的 ONU 终端产品，为 FTTH、FTTR、FTTM 以及 POL 提供了独立保障和产品支撑。


# 自研力打造数智网络，助力广电业务抢占先机 #

方向有了，技术和产品有了，还需要应用落地。自研力打造的数智网络，要在应用业务上打出“创新”牌，才有可能弯道超车。我们希望可以为广电弯道超车、跨越式发展献一份力。


我们认为第一张牌是“创新C端业务”。5G融合新通信，在数据通道的加持下，可提供原生的创新音视频融合服务，利用智能语音、人脸识别及音视频识别、处理、合成媒体技术，实现创新业务。例如实时话音翻译，通过媒体云的AI智能语音识别，识别双方实时话音，调用媒体云翻译组件，翻译为语言文本；再调用媒体云的AI视频生成能力，将语言文本以字幕的形式推送至客户的终端屏幕上。可服务于商务人士、听障人士。


第二张牌是“创新专网方案”。我们为2B 业务提供了 5G 工业融合专网解决方案，为车载、船载场景提供了智简融合的定制化部署方案，核心网可以在1块单板、1台服务器上实现部署，提供端到端、即插即用的云网融合应用。全面支持信创要求，基于 ARM 的核心网解决方案，未来演进支持 RISC-V。震有提供集成网管实现动态业务检测，问题处理与业务控制，通过预置策略实现业务快速恢复，降低行业专网的核心网运维难度，解决园区、工厂等行业客户无法专门配置专业运维人员的突出问题。


第三张牌是“打差异化”。5G 深入千行百业，带来了极为丰富的业务场景和需求，例如野外高带宽通信场景，需要将核心网部署在低轨上；例如高密度接入场景下的 5G 直播，需要考虑在高负载大用户量接入的网络中，保证 5G 直播需要的带宽和 QoS；例如工厂的生产线，大量的终端、汽车等需要海量数据的预安装和测试，需要确保这些终端能像接入 WiFi 一样简单、平滑接入高质量 5G 网络。只有满足行业的特殊需求，才能让5G真正创造价值。震有以自研力为基础，构建起为各行业、各省客户提供差异化的定制能力，满足客户需求。


最后是“切入复杂工业场景”。复杂工业场景是产业数智革命的关键所在，千兆光网具有传输带宽大、抗干扰能力强的特点，是切入工业复杂场景的必选项。在传统的物联网和工业互联网中，互联互通繁锁；有源产品功耗较大，运维成本高；IOT 产品复杂，完整方案缺乏。震有打造“CROSS”能力网络，基于新一代云化控制器系统NuMax Cloud 平台，提供具有联接（Connectivity）、实时（Real-time）、数据优化（Optimization）、智能（Smart）、安全与隐私保护（Security）五大能力的智慧光网。当然，还有工业 PON 多种等级的保护倒换方案，以满足安全性的设计考量等等。


铸造数智网络要靠自研力，服务数字中国建设更要靠自研力。在此感谢主办方提供了这次分享的机会，也衷心欢迎震有的老朋友、新朋友们莅临深圳总部指导和交流！谢谢大家！
